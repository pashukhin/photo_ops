# Sessions

Each work session has a brief in this directory, numbered sequentially in
chronological order: `001_*`, `002_*`, `003_*`, and so on. The number gives
the order; the descriptive name (e.g. `_fortification_review`,
`_agent_working_conditions`) makes the session's purpose obvious. There is no
separate per-type numbering — product feature work, reviews, and
agent-tooling sessions all share one flat sequence.

Briefs are human-readable summaries; the accepted design and plan for each
session live under `docs/superpowers/specs` and `docs/superpowers/plans`.
